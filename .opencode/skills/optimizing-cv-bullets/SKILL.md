# CV Bullets Optimizer

## When to use this skill

- User wants to improve existing CV bullet points
- User needs to create new bullet points from job duties
- User wants to make bullets more impactful/quantifiable
- User needs ATS-optimized bullets
- User wants to tailor bullets for specific job

## Bullet Formula

Use this structure for maximum impact:

```
[Acción] + [Qué] + [Resultado medible/impacto] + [Herramienta/Tecnología]
```

### Examples transformation:

**Weak:** "Implemented payments with Stripe"
**Strong:** "Implemented Stripe payments, processing $50K/month with 99.9% uptime"

**Weak:** "Worked with React"
**Strong:** "Built React components reducing load time by 40%"

**Weak:** "Managed database"
**Strong:** "Optimized MySQL queries reducing response time from 2s to 200ms"

## Action Verbs by Category

### Leadership/Management
- Led, Directed, Coordinated, Managed, Mentored, Trained

### Technical/Development
- Developed, Built, Implemented, Designed, Architected, Refactored

### Results/Impact
- Increased, Reduced, Improved, Saved, Achieved, Delivered

### Collaboration
- Collaborated, Partnered, Communicated, Liaised

### Problem Solving
- Resolved, Troubleshot, Diagnosed, Optimized, Automated

## Instructions

1. **Read current bullets** or get raw job duties
2. **Apply STAR improvement**:
   - Situation: What was the challenge?
   - Task: What was your responsibility?
   - Action: What did YOU do specifically?
   - Result: Quantify the outcome

3. **Add metrics** wherever possible:
   - Percentage improvements
   - Dollar amounts saved/earned
   - Time saved
   - Users impacted
   - Scale (users, requests, data)

4. **Include tools/tech** at end of bullet
   - Use **bold** for tech names in LaTeX

5. **Keep to 2 lines max**, 1 ideal

## ATS Optimization

- Use standard job titles
- Include keywords from job description
- Avoid: tables, graphics, headers/footers, columns
- Use standard section headers
- Plain text friendly

## Common Mistakes to Fix

| Bad | Good |
|-----|------|
| "Responsible for..." | Start with action verb |
| "Worked on..." | "Delivered..." |
| "Knew..." | Show impact |
| Too many duties | 3-4 top achievements |
| No numbers | Add metrics |
| Generic | Be specific |

## Output Format

Return optimized bullets in this format:

```
[Company Name] - [Role]
- Optimized bullet 1 with **technology**
- Optimized bullet 2 with **technology**
- Optimized bullet 3 with **technology**
```

## Quick Fix Checklist

- [ ] Starts with action verb
- [ ] Shows measurable result
- [ ] Includes technology/tool
- [ ] Max 2 lines
- [ ] No "I" or "my"
- [ ] No passive voice
